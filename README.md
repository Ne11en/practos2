# Консольное приложение C#

Простое консольное приложение на C# с демонстрацией функций.

## Требования

- .NET 6.0 SDK или выше

## Запуск приложения

1. Откройте терминал в папке проекта
2. Выполните команду:
```
dotnet run
```

## Сборка проекта

Для сборки проекта выполните:
```
dotnet build
```
